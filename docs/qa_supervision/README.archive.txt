# 🛡️ 质量保证总控台 (QA Supervision)

> 这里是本项目的“数据法庭”。所有 AI Agent 对底层架构的修改，其优劣最终都只能由本目录下严格的**对比数据**与**可视化路由动画**来定性。

## 目录结构
- `benchmark_tracking.md`：核心跟踪文档。每次版本迭代的 SABRE vs AI 对比量化跑分都在这里统一登记。所有进度一目了然。
- `visual_reports/`：(未来由 AI 自动生成) 存放拓扑连接态与路由演示 GIF 图。

## 交互契约
- 执行 `/supervised-dev` 工作流时，AI 只有在局部的冒烟测试（Smoke Test）在 `benchmark_tracking.md` 中拿到了打倒基线算法的数据后，才会被用户批准进入长期、宏大规模的挂机训练验证。
- 一切虚无缥缈的技术堆砌在此绝迹，唯有**性能提升**与**可解释的画图**才是通过验收的唯一标准。
