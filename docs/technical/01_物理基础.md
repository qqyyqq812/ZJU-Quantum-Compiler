# 量子电路编译 — 物理基础知识库

## 一、量子比特与量子门

### 量子比特 (Qubit)
- 经典比特只能是 0 或 1；量子比特可以处于叠加态：|ψ⟩ = α|0⟩ + β|1⟩
- |α|² + |β|² = 1（归一化条件）

### 常用量子门
| 门 | 符号 | 类型 | 作用 |
|----|------|------|------|
| Pauli-X | X | 单比特 | 比特翻转（类似经典 NOT） |
| Hadamard | H | 单比特 | 创建叠加态 |
| CNOT | CX | 双比特 | 受控非门，量子纠缠的基础 |
| SWAP | SWAP | 双比特 | 交换两个量子比特的状态，= 3个CNOT |
| Toffoli | CCX | 三比特 | 受控受控非门 |

### 关键概念：CNOT 门的成本
- CNOT 是双量子比特门，**只能在物理上相邻的比特之间执行**
- 1 个 SWAP = 3 个 CNOT → SWAP 门的成本很高
- **减少 SWAP/CNOT 门数量是本项目的核心优化目标**

## 二、量子电路的表示

### DAG（有向无环图）表示
- 量子电路可以表示为 DAG：节点 = 量子门，边 = 量子比特（数据依赖）
- 这与经典 EDA 中的逻辑网表表示**完全类比**
- Qiskit 的 `DAGCircuit` 类就是这种表示的标准实现

### Coupling Map（耦合图）
- 描述物理量子芯片上哪些比特之间可以直接执行双比特门
- 例如 IBM Eagle (127 qubit) 的 Coupling Map 是一个稀疏图，每个比特只与 2-3 个邻居相连
- **类比经典 EDA**：Coupling Map ≈ FPGA 的布线资源 / ASIC 的金属层连接约束

## 三、量子电路编译的三个阶段

```
逻辑量子电路 → ① 门分解 → ② 初始映射 → ③ 路由(SWAP插入) → 物理量子电路
```

1. **门分解 (Gate Decomposition)**：将高级门分解为芯片原生门集合（类似逻辑综合）
2. **初始映射 (Initial Mapping)**：逻辑比特 → 物理比特的初始分配（类似 Placement）
3. **路由 (Routing)**：插入 SWAP 门使所有双比特门满足连通性约束（类似 Routing）

> 本项目聚焦**阶段 ② 和 ③**，即映射与路由的 AI 优化。
