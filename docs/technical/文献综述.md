# 量子电路编译路由算法文献综述

> 本文综述了量子电路编译中**路由（Routing）**问题的主要研究方法，涵盖启发式搜索、强化学习和图神经网络三大技术路线。通过对 8 篇核心文献的系统分析，阐明了本项目的技术定位与创新点。

---

## 一、问题背景

### 1.1 量子电路编译问题

量子计算机的物理量子比特之间并非全连接。大多数超导量子处理器（如 IBM 的 Heavy-Hex 拓扑）只允许相邻物理比特之间执行双比特门（如 CNOT）。因此，逻辑量子电路到物理量子芯片的编译过程中，需要通过插入 **SWAP 门** 来搬运数据，使远距离的量子比特能够交互。

> **类比**：传统编译器需要做寄存器分配（将变量映射到有限的物理寄存器），量子编译器需要做 **比特映射与路由**（将逻辑比特映射到物理比特，并动态调整映射关系）。

### 1.2 核心优化目标

| 指标 | 说明 | 重要性 |
|------|------|--------|
| **SWAP 门数** | 插入的额外 SWAP 门数量 | ⭐⭐⭐ 最关键 |
| **电路深度** | 编译后电路的总层数 | ⭐⭐ 影响运行时间 |
| **编译时间** | 编译器本身的运行时间 | ⭐ 大规模电路时重要 |

每个 SWAP 门分解为 3 个 CNOT 门，每个 CNOT 门在当前硬件上的保真度约 99%。因此，SWAP 门数直接决定了量子电路的整体保真度，是路由优化的首要目标。

## 二、启发式搜索方法

### 2.1 SABRE [1]

**论文**：*Tackling the Qubit Mapping Problem for NISQ-Era Quantum Devices*  
**作者**：Gushu Li, Yufei Ding, Yuan Xie  
**会议**：ASPLOS 2019

SABRE（SWAP-based Bidirectional heuristic search）是目前业界最广泛使用的量子电路路由算法，已集成于 IBM Qiskit 框架。其核心思想是：

1. **前沿层（Front Layer）驱动**：每一步关注当前可执行的双比特门集合（前沿层），选择能最大限度缩短前沿门物理距离的 SWAP。
2. **双向遍历（Bidirectional Traversal）**：先从前到后遍历电路一次得到初步映射，再反向遍历一次优化初始映射，循环数轮取最优。
3. **衰减因子（Decay Factor）**：对已涉及 SWAP 的比特施加衰减惩罚，避免 SWAP 过度集中于某些比特，从而均衡电路深度。

**关键贡献**：
- 首个实用的面向 NISQ 设备的路由算法
- O(n²) 时间复杂度，可扩展至数百比特
- 在 Qiskit 中作为默认路由方法，成为后续所有工作的 **baseline**

**局限性**：完全基于贪心启发式，缺乏全局规划能力；在高连通性电路中容易陷入局部最优。

### 2.2 LightSABRE [2]

**论文**：*LightSABRE: A Lightweight and Enhanced SABRE Algorithm*  
**作者**：IBM Quantum 团队  
**发表**：arXiv 2024

LightSABRE 是 SABRE 的工程级增强版本，主要改进包括：

1. **Rust 高性能实现**：将核心路由循环从 Python 迁移至 Rust，实现约 **200 倍** 的运行加速。
2. **相对评分机制**：不再使用绝对距离，而是比较 SWAP 前后的距离变化比率，提升 SWAP 选择的区分度。
3. **多次随机试验**：以不同初始映射运行多次路由，取最优结果，利用统计手段弥补贪心的缺陷。
4. **关键路径分析**：识别电路中的关键路径（最长依赖链），优先保障关键路径上的门可执行。

**核心成果**：在标准基准集上，SWAP 门数平均减少 **18.9%**，编译时间缩短约 200 倍。这一改进已集成至 Qiskit 1.0+，是目前 SABRE 族的最强版本。

## 三、强化学习方法

### 3.1 RL for Quantum Circuit Compilation [3]

**论文**：*Qubit Routing using Reinforcement Learning*  
**发表**：arXiv 2020 / NeurIPS-W 2021

该工作是强化学习应用于量子路由的**开山之作**。核心设计：

- **状态表示**：以映射矩阵 + 前沿门距离向量作为 MDP 的状态空间
- **动作空间**：所有可行 SWAP 边 + PASS 动作
- **奖励函数**：执行门给正奖励，SWAP 给负惩罚，前沿距离缩短给额外奖励
- **算法**：使用 PPO 进行策略优化

**局限性**：
- 仅使用全连接 MLP 编码状态，无法捕获拓扑结构信息
- 训练仅在小规模电路上验证（5-10 比特）
- 泛化能力不足，换拓扑需要重新训练

**意义**：奠定了 RL+量子路由的基本 MDP 框架，后续所有 RL 方法均以此为基础。

### 3.2 AlphaRouter [4]

**论文**：*AlphaRouter: Quantum Circuit Routing with Reinforcement Learning and Tree Search*  
**作者**：Wei Tang, Yiheng Duan, Yaroslav Kharkov 等  
**会议**：IEEE QCE 2024

AlphaRouter 将 AlphaGo 的 **MCTS + RL 混合搜索** 思想引入量子路由：

1. **策略网络**：训练一个 RL 策略网络为每个 SWAP 候选打分
2. **价值网络**：预测当前状态下完成路由还需多少步
3. **蒙特卡洛树搜索**：以 RL 策略为先验，通过 MCTS 在搜索树中探索最优 SWAP 序列
4. **自我对弈（Self-Play）**：策略网络与 MCTS 相互提升

**核心成果**：相比 SABRE，路由开销减少 **20%**，在多种拓扑上均有提升。

**局限性**：MCTS 搜索在推理时计算量大，编译速度较慢（约 10-100 倍于 SABRE）。

### 3.3 PPO (Proximal Policy Optimization) [5]

**论文**：*Proximal Policy Optimization Algorithms*  
**作者**：John Schulman, Filip Wolski, Prafulla Dhariwal 等  
**发表**：arXiv 2017

PPO 是当前最广泛使用的策略梯度 RL 算法，核心思想：

- **裁剪目标函数**：限制新旧策略的 KL 散度，防止策略更新过大导致训练不稳定
- **优势估计（GAE）**：使用广义优势估计平衡偏差与方差
- **多 epoch 更新**：同一批数据可以更新多次，提升数据效率

PPO 是**所有 RL 量子路由工作**（包括本项目）的核心训练算法，因其稳定性和超参数鲁棒性，已成为 RL 领域的标准工具。

## 四、图神经网络方法

### 4.1 ZX-RL [6]

**论文**：*Reinforcement Learning Based Quantum Circuit Optimization via ZX-Calculus*  
**发表**：arXiv 2023 / Quantum 2024

ZX-RL 的核心创新在于将 **ZX-图** 的化简和 **GNN + PPO** 结合：

1. **ZX-图表示**：将量子电路转换为 ZX-图（一种图论表示），利用 ZX-演算的化简规则作为 RL 动作空间
2. **GNN 编码器**：使用图注意力网络（GAT）编码 ZX-图的结构特征，捕获节点间的依赖关系
3. **PPO 策略**：基于 GNN 提取的特征，用 PPO 选择最优化简规则

**关键发现**：
- GNN 编码在 **80Q 以上** 电路仍然有效，证明了图结构表示的可扩展性
- 泛化能力：在小规模电路上训练的策略可迁移到大规模电路

**对本项目的启发**：验证了 GNN + PPO 框架对量子电路优化任务的有效性。

### 4.2 GNN for Quantum Circuits [7]

**论文**：*Quantum Circuit Optimization with Deep Reinforcement Learning*  
**作者**：Thomas Fösel 等  
**发表**：ML:Science and Technology 2021

该工作系统研究了如何用 GNN 编码量子电路的图结构：

- **双图架构**：同时编码电路的 DAG 结构（逻辑依赖）和芯片的物理拓扑图
- **消息传递**：通过多轮消息传递聚合邻域信息，捕获长距离依赖
- **特征设计**：节点特征包含门类型、比特角色（控制/目标）、执行状态等

**核心贡献**：为 GNN 在量子编译中的应用奠定了理论基础，提出了量子电路图特征工程的标准范式。

## 五、训练技巧

### 5.1 Q-CurL (Quantum Curriculum Learning) [8]

**论文**：*Quantum Curriculum Learning*  
**作者**：Quoc Hoan Tran 等 (Fujitsu Research)  
**发表**：arXiv 2024

Q-CurL 将**课程学习**理念引入量子机器学习：

1. **渐进式难度调度**：先在简单任务（小量子比特、浅电路）上训练，逐步增加难度
2. **密度比自适应**：基于量子数据密度比动态调整课程进度
3. **噪声鲁棒性**：课程学习显著提升了模型对量子噪声的鲁棒性

**对本项目的启发**：本项目实现了 5 阶段课程学习调度器（3Q→20Q），与 Q-CurL 的设计理念一致。V7.1 版本进一步加入了自适应宽限机制，当模型卡住时自动放宽阈值。

### 6.2 本项目的技术定位

本项目（V7.1 量子电路 AI 编译器）融合了上述多条技术路线的优势：

```
                 ┌─────────────┐
                 │  SABRE [1]  │  ← Baseline 对手
                 └──────┬──────┘
                        │ 启发式搜索框架
                        ▼
        ┌───────────────────────────┐
        │  RL for Quantum [3]       │  ← MDP 框架设计
        │  + PPO [5] 训练算法       │
        └───────────┬───────────────┘
                    │ 加入图结构建模
                    ▼
        ┌───────────────────────────┐
        │  GNN Encoder [6,7]        │  ← 芯片拓扑图编码
        │  (GraphSAGE 3层)          │
        └───────────┬───────────────┘
                    │ 加入课程学习
                    ▼
        ┌───────────────────────────┐
        │  ★ 本项目 V7.1 ★          │
        │  PPO + GNN + 课程学习 [8] │
        │  + Action Masking         │
        │  + Edge-Scoring Attention │
        │  + Checkpoint Resilience  │
        └───────────────────────────┘
```

### 6.3 本项目的创新点

1. **Edge-Scoring Attention 动作头**：不同于传统方法将所有 SWAP 候选编码为固定维度向量，本项目使用基于边的注意力打分机制，直接为每条 SWAP 边独立评分，解决了维度灾难和动作排列无序性问题。

2. **V7.1 自适应课程学习**：结合 Q-CurL 的渐进学习理念，实现了 5 阶段课程调度器（3Q→20Q），并加入了自适应宽限机制 —— 当模型在某阶段停留过久时，自动放宽阈值以避免训练停滞。

3. **工业级容错训练管线**：通过 checkpoint 定期保存、OMP 线程控制、日志监控等机制，确保长时间训练在 CPU-only 环境下的稳定性。
