# M1 阶段总结：领域知识建设

> 完成时间：2026-03-21

## 完成内容

### 1. Qiskit 工具链入门
- 安装 Qiskit 2.3.1 + Aer 0.17.2 模拟器
- 成功创建并模拟 Bell 态电路（|00⟩=50.2%, |11⟩=49.8%）
- 文件: `sandbox/01_qiskit入门/bell_state.py`

### 2. Coupling Map 拓扑分析
- 创建并分析 5 种拓扑：线性、环形、3×3网格、重型六边形(19q)、全连接
- 可视化对比图 + JSON 数据输出
- 关键发现：heavy_hex 19q 拓扑直径=8，平均路径=3.92 → CNOT(q0, q18) 可能需要 7 个 SWAP
- 文件: `sandbox/01_qiskit入门/coupling_map_viz.py`

### 3. SABRE 算法精读
- 精读 Li et al., ASPLOS 2019
- 详细分析：算法流程、启发式函数 H、双向搜索机制
- 总结 4 大局限性 → 确认GNN/RL可超越的理论基础
- 文件: `docs/technical/03_SABRE精读.md`

### 4. Transpiler 实战演示
- 在 3 种拓扑（线性/环形/全连接）上编译同一电路
- 对比编译前后 CNOT 数和电路深度
- 文件: `sandbox/01_qiskit入门/transpiler_demo.py`

### 5. GNN/RL 文献综述
- 分析 5 篇核心论文（包括复杂度证明、DQN路由、Deep RL优化、GAT电路编码、GNN+PPO端到端）
- 确定技术选型：GAT + PPO + 三元组 State
- 文件: `docs/technical/05_文献综述.md`
