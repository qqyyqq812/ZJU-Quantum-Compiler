# M2 阶段总结：基线实现与评估体系

> 完成时间：2026-03-22

## 完成内容

### 1. 基准电路集 (`src/benchmarks/circuits.py`)
- QFT、Grover、QAOA、Random 四种标准量子算法
- 支持 5-20 qubit 参数化生成
- `get_benchmark_suite()` 一键生成全套基准

### 2. 拓扑集 (`src/benchmarks/topologies.py`)
- 7 种标准拓扑：linear, ring, grid, heavy_hex, full + IBM Eagle(127q) + Google Sycamore(54q)
- `get_topology(name)` 统一接口
- `get_topology_info()` 自动分析属性

### 3. 评估框架 (`src/benchmarks/evaluate.py`)
- `CompileResult` 数据类：depth, cx, swap, time + 开销比
- `evaluate_compiler()` 统一评估入口
- JSON 持久化 + Markdown 对比表生成

### 4. DAG 操作模块 (`src/compiler/dag.py`)
- `CircuitDAG` 从 QuantumCircuit 构建 DAG
- 前沿层提取、门执行、SWAP 映射更新
- `to_networkx()` + `get_node_features()` 为 M3 GNN 提供接口
- `get_interaction_graph()` 提取比特交互图

### 5. SABRE 基线 (`results/baseline_sabre.json`)
- 4 种电路 × 3 种规模 × 7 种拓扑 × SABRE O1 = **60 次编译**
- 平均 CX 开销: **123.5%**
- 平均深度开销: **327.9%**
- 平均编译时间: **8.9 ms**

## 关键基线数据洞察

- **拓扑影响显著**: grid 拓扑 SWAP 开销远低于 linear
- **QFT 最难路由**: 15q QFT 在 linear 拓扑上需要 222 个额外 SWAP
- **QAOA 拓扑友好**: 环形结构匹配 ring 拓扑时零 SWAP 开销
- **AI 编译器的机会**: 平均 123.5% CX 开销意味着有大量优化空间
