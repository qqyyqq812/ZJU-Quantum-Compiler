### 消融实验结论
| 配置 | Avg Reward | Avg SWAPs |
|------|-----------|-----------|
| baseline (100ep, lr=3e-4) | 29.1 | 33.5 |
| higher_penalty (swap=-0.5) | 20.9 | 36.7 |
| lower_lr (lr=1e-4) | 26.9 | 40.9 |
| **more_training (200ep)** | **30.3** | **26.9** |

→ 更多训练显著降低 SWAP 数，更高惩罚反而不利（Agent 过于保守）
