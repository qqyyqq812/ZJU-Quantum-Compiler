# 阶段一至三（V1-V3）总结报告

在过去的三阶段开发中，我们从零开始，逐步构建并深度优化了一个基于深度强化学习（PPO）的**量子电路智能路由（Quantum Circuit Routing）编译器框架**。该研究旨在通过人工智能自动学习芯片物理拓扑约束下的 SWAP 门最佳插入策略，以替代传统的启发式算法（如 SABRE），从而极大地降低量子程序在易噪量子硬件上的执行时间与门数量开销。

## 1. 核心挑战与问题定义 (Problem Formulation)
量子计算设备（如超导量子芯片）受到通信拓扑（Coupling Map）的严重限制，并非所有量子比特之间都具备直接相互作用（执行两比特门如 CNOT）的物理连接。当程序中要求对不相邻的逻辑量子比特执行指令时，编译器必须插入辅助的 SWAP（交换）门将信息移动到相邻位置。然而，由于 SWAP 门是由3个 CNOT 门拼接而成，其极高的固有错误率会对电路的计算保真度造成毁灭性打击。
因此，如何**最小化插入的 SWAP 门总数**是一个典型的 NP-Hard 优化问题，而设计具有极强泛化能力和适应性的路由算法，是本项目的终极追求目标。

## 2. 阶段性研发过程与突破 (Methodology & Milestones)

### 第一阶段（Phase 1: V1）：基础全栈基建映射
在初期阶段，项目聚焦于打通算法侧与 Qiskit 编译栈的核心通道。
- **环境构建**：利用 Gymnasium 为底层，构建了标准的马尔可夫决策过程（MDP）。在此 MDP 中，动作空间被映射为由物理拓扑支持的所有可能的 SWAP 操作。
- **模型设计**：初步应用了基于 MLP 的 Actor-Critic 架构的 PPO 稳定算法。
- **阶段性产出**：跑通了整个 AI 路由器的训练链路测试，但因特征贫乏及奖励机制极为简单（仅凭最后一步稀疏奖励），模型对电路和拓扑缺乏认知。测试结果证明，V1 距离 SABRE 具有接近 6 倍差距的设计鸿沟。

### 第二阶段（Phase 2: V2）：课程学习与密集奖励设计
为解决稀疏奖励情况下的探索困难，V2 发起了深入的工程重构：
- **距离收缩密集特征（Distance-Based Feature）**：通过前沿指令层（Front Layer）至所有候选物理位置的最短路径，构造了状态观测向量（Observation），让多层感知机（MLP）具有了感知物理限制的空间视觉。
- **势能差奖励引擎（Potential-based Reward）**：使用插入 SWAP 先后的前沿位点距离差值作为 Dense Reward。
- **课程学习机制（Curriculum Learning）**：从最基础的深度为1、两比特电路开始，逐步向 QFT、QAOA 过渡，克服了一开始直接求解大电路无法收敛的问题。
- **阶段性产出**：使得大范围拓扑的 SWAP 比率急剧下降，`linear_5` 等微拓扑下的评价比率收缩至 **2.11x**。

### 第三阶段（Phase 3: V3）：行动掩码约束与视野延展
针对模型后期偶发的自环震荡（无效地来回进行 SWAP）以及不理智行为：
- **绝对掩码防御（Action Masking）**：通过严密的解析计算，屏蔽所有的不向目标门方向移动的无效 SWAP 边。将 RL 大海捞针的探索树修剪为了精准优化树。
- **超前感知预判（Look-Ahead Fronts）**：突破局部阻塞效应，从仅观察底层阻断门扩大至计算第二甚至第三依赖层门（Extended Fronts）的总位移预期。
- **GNN 的结构化拓展尝试**：实验性地在框架中集成了 Graph Attention Network（GAT），以验证空间拓扑提取的可行性。长效训练结果表明：虽然在极其复杂的状态泛化上受阻于节点死锁与拟合坍缩，但基础的 MLP+Masking 模型（10k Episodes下）已经成功逼近并在部分主流拓扑下（如 3x3 Grid QFT 电路）反杀 SABRE。
- **阶段性产出**：确立了以 1.63x ~ 1.86x 为基准的卓越黄金比率（超越并战平顶级工业算法），确立了 V3(MLP+Masking+LookAhead) 的最优态。

## 3. 面向未来的优化愿景 (Implications)
前三阶段的工作成功用 RL 框架跑通并大幅度赶超了古典编译器。基于目前的架构数据，我们在大规模纠缠多图（大节点度网络）上已经出现表征瓶颈与过拟合效应，同时，暂未兼顾串扰扰动（Crosstalk）等物理底层的噪声因素。这两大现实痛点将指导我们设计下一代 V4 编译器的基础架构（如引入 MCTS 与噪声感知）。
