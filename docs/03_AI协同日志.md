## 2026-03-21 — M1 领域知识自动化建设

**问题**: 从零建立量子电路编译的知识体系
**Prompt 策略**: 使用 `/quantum-dev` 工作流，让 AI 全自动执行：
1. 安装 Qiskit 并跑通第一个量子电路
2. 分析 5 种芯片拓扑的 Coupling Map 性质
3. 精读 SABRE 论文并撰写结构化笔记
4. 文献综述 5 篇 GNN/RL+量子编译论文
5. 编写 10 项自动化测试

**结果**:
- Qiskit 2.3.1 成功安装，Bell 态验证通过
- 5 种拓扑量化对比（直径、平均路径、SWAP 需求）
- SABRE 局限性分析 → 确认 GNN/RL 的改进方向
- 文献综述确定技术选型：GAT + PPO
- 全部 10 项 pytest 测试通过

**AI 自我纠错**: Qiskit 2.x 使用 rustworkx 替代 networkx 作为内部图库，初始代码出现 API 不兼容错误。AI 自动诊断并修复，无需人工干预。

**反思**: 全自动工作流显著提升效率 — M1 全部任务在单次对话中完成，用户零决策参与。
