## 【已解决的核心问题】

### 1. Soft Action Mask 乒乓死循环问题
- **现象**: 放开距离约束后，AI 在两个相邻节点间无限来回互换 (SWAP 从 26 暴涨至 443)
- **根因**: 模型无记忆，退一步后立即被"距离缩减奖励"吸引回原位
- **解法**: 在 `env.py` 的 `step()` 中植入 Ping-Pong Penalty (-5.0)。模型在训练中自由触发惩罚并学会避免（而非在 mask 层物理屏蔽）
- **关键代码**: `env.py:154-164` — `if action in self.tabu_list: reward += self.penalty_tabu`

### 2. SOTA 方案对比实验 (Ablation Study)
- **Tabu 推理拦截**: 无效（模型绕 4 步圈子规避 3 步黑名单）
- **Beam Search**: 崩溃（Qiskit DAG 深拷贝内存泄漏，0.1s 内异常退出）
- **结论**: 纯推断补丁不可行，必须在训练层重塑神经元权重
- **数据**: 见 `docs/qa_supervision/benchmark_tracking.md`

### 3. 断点续传硬编码 Bug
- **原始问题**: `run_train_v9_20Q.sh` 写死了 `checkpoint_ep5000.pt`，训练超过该点后丢失进度
- **修复**: 改为 `$(ls -v ${SAVE_DIR}/checkpoint_ep*.pt | tail -n 1)` 自动搜寻最新存档

### 4. 拓扑可视化引擎 (M2 交付件)
- **文件**: `src/visualization/topology_visualizer.py` (~300 行)
- **能力**: IBM Tokyo 20Q 精确物理布局 → 暗色科研主题渲染 → 路由逐帧捕获 → GIF 动画合成
