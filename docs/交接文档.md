## 核心架构

### 观测编码 (V9 拓扑无关)
- **obs_dim = 291**，与物理比特数 N 无关
- 构成: SWAP 边特征 (240) + 前沿门距离 (20) + Look-ahead (20) + 统计量 (10) + 进度 (1)
- 代码: `src/compiler/env.py`

### 策略网络 (V7 GNN Actor-Critic)
- 3 层 GraphSAGE 编码器 + Edge-Scoring Attention + 独立 Critic
- 代码: `src/compiler/policy.py`

### 训练管线
- PPO + 5 阶课程学习 (3Q→5Q→5Q→10Q→20Q)
- LR 余弦退火 + 熵系数线性衰减 + Early Stopping
- 代码: `src/compiler/train.py`, `src/compiler/curriculum.py`

### 推断引擎 (V8)
- MTx100 为核心策略（100 次随机初始映射 + 贪心推理取最优）
- 代码: `src/compiler/inference_v8.py`
